Notificaciones según su estado

Este reporte muestra la cantidad de notificaciones cumplidas y no cumplidas para elaborar un gráficos de pastel que permite visualizar la efectividad de las notificaciones enviadas. Esto es útil para evaluar el rendimiento del sistema de notificaciones y tomar decisiones informadas sobre posibles opciones de mejora o alternativas diferentes para la comunicación de las notificaciones con los usuarios. Puedes exportar este reporte a PDF para su impresión o almacenamiento.
